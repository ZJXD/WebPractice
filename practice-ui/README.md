# practice-ui

基于 Vue 的前端练习项目

## 项目构建过程

### Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

### Customize configuration

参考 [Vue 官方脚手架配置](https://cli.vuejs.org/config/).
